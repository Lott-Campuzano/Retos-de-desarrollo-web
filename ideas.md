# ¿Quieres contribuir?

## Instrucciones para añadir ideas al banco de ideas:
1. Crea un fork de este proyecto.
2. Elige este archivo y haz clic en el ícono de lápiz para editarlo.
3. Agrega tus idea de proyecto, para poderla desarrollar u añadir al repositorio.
4. Asegúrate de incluir una breve descripción de tu idea, así como tu usuario de Github (Incluyendo el link a tu perfil).
5. Una vez que hayas terminado, haz clic en "Proponer cambios" y luego en "Crear solicitud de extracción".
6. ¡Gracias por contribuir!

## Instrucciones para añadir ideas al directorio comunitario:
1. Crea un fork de este proyecto.
2. Dirigete al directorio "directorio comunitario" y crea un subdirectorio para tu idea.
3. Dentro del subdirectorio, carga todo tu proyecto.
4. Despues crea un archivo README.md 
5. Dentro de este archivo describe tu idea de proyecto y añade tu usuario de Github (Incluyendo el link a tu perfil).
6. Una vez que hayas terminado, haz clic en "Proponer cambios" y luego en "Crear solicitud de extracción".
7. ¡Gracias por contribuir!
